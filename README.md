# Toolkit Data

This repository contains the toolkit-data package used by the CivicPlus Toolkit project.

The single source of truth for loader rules and helper scripts is located at:
`specs/CivicPlus_Toolkit_Upgraded_Master_Specification.pdf`

## Quick start
- `index.json` contains a machine-readable index of the package.
- `INDEX.md` contains a human-friendly listing and links.
